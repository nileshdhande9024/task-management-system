# Task Management System

## Tech Stack
- React JS
- Spring Boot
- MySQL

## Features
- Add Task
- Update Task
- Delete Task
- View Tasks

## Description
This is a full-stack CRUD application built using React and Spring Boot.
It allows users to manage tasks with basic operations like add, update, delete, and view.

## Project Structure
task-management-system/
├── frontend/
└── backend/

## How to Run

### Backend (Spring Boot)
- Open in Eclipse
- Run as Spring Boot Application

### Frontend (React)
- Open in VS Code
- Run:
  npm install
  npm start
